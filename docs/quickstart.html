Sensorberg SDK

Quickstart

1. Create an account

The first thing you need to do to get started with Firebase is sign up for a free account.1

2. CocoaPods setup

The easiest way to integrate the iOS SDK is via CocoaPods 2. If you're new to CocoaPods, visit their getting started documentation.

cd your-project-directory    
pod init

Once you've initialized CocoaPods, just add the Sensorberg Pod to your Podfile:

pod 'SensorbergSDK', '~> 2.0'
pod 'tolo', '~> 1.0'

3. Gettings started in Xcode

Objective-C

Include the Sensorberg SDK header in your app to get all the needed classes: #import <SensorbergSDK/SensorbergSDK.h>

Swift

You can use the Sensorberg SDK in your Swift class by simply importing the module:

import SensorbergSDK
